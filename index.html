Інструкція, яка має бути найпершим рядком HTML-документа. Вона повідомляє браузеру, що сторінка написана на сучасному стандарті HTML5 , переводячи його у стандартний режим для коректного відображення макета.

Щоб вказати на початку документа потрібно ввести !, і натиснути Enter Розгорнуться стандатрні налаштування. 


!DOCTYPE html
Example:

!DOCTYPE html

Атрибут у HTML, який використовується для визначення мови вмісту елемента, зазвичай встановлюється в тегу html. Він допомагає скрінрідерам правильно вимовляти текст, а пошуковим системам — коректно індексувати та надавати результати . 


  html lang="uk" html
Example:
html lang="uk" html

Контейнер для метаданих HTML-документа, який розміщується між тегами html та body. Він містить інформацію про сторінку, яка не відображається користувачам безпосередньо у вікні браузера. Усередині head зазвичай розміщують title, посилання на CSS-файли (link), JavaScript-скрипти, інформацію про кодування meta charset="utf-8" та SEO-дані.

Тег meta — це самозакривний тег, який розміщується всередині head і використовується для надання метаданих (додаткової інформації) про HTML-документ. Ця інформація не відображається безпосередньо, але використовується браузерами, пошуковими системами та іншими вебсервісами. Тег meta не має закриваючого тегу.

name="viewport": Це повідомляє браузеру, що метадані стосуються області перегляду (Viewport) — видимої області сторінки.
Цей тег є критично важливим для адаптивного дизайну.

content="width=device-width, initial-scale=1.0": Це сама інструкція для області перегляду, що складається з двох частин:

width=device-width: Встановлює ширину області перегляду рівною фізичній ширині пристрою (замість використання ширини робочого столу за замовчуванням).
initial-scale=1.0: Встановлює початковий коефіцієнт масштабування сторінки на 100%. Це гарантує, що при першому завантаженні сторінка не буде масштабована браузером (наприклад, щоб вона виглядала меншою на мобільному телефоні).

Тег p — це парний блочний елемент який займає всю доступну йому ширину, який використовується для створення текстових абзаців; він автоматично додає вертикальні відступи (margins) зверху та знизу, потрібно обнуляти стилі для усунення мерджинів. Завжди починається з нового рядка і може містити лише текст або рядкові елементи (наприклад, a чи strong).

Тег ul (unordered list) — це парний блочний елемент для створення маркованих списків, де кожен пункт за замовчуванням позначається крапкою, а тег li (list item) розміщується всередині нього для визначення окремих пунктів цього списку. Така структура дозволяє логічно групувати пов'язані елементи, автоматично додаючи до них лівий відступ для візуального виділення.

Тег s — це парний рядковий елемент, який використовується для відображення тексту, що більше не є актуальним, точним або релевантним. Браузери за замовчуванням відображають вміст цього тегу закресленим. Це суто візуальне закреслення, яке не слід плутати з тегом del, що вказує на видалення документальної правки.

Тег u — це парний рядковий елемент, який використовується для візуального підкреслення тексту. У сучасній семантиці HTML він застосовується для позначення тексту, який має особливий нетекстовий опис (наприклад, для виділення орфографічних помилок або власних назв у китайській мові), хоча за замовчуванням браузери просто малюють під ним лінію. Слід бути обережним із його використанням, щоб користувачі не плутали підкреслений текст із посиланням.

Тег sub — це парний рядковий елемент, який використовується для відображення тексту у нижньому індексі. Він рендериться шрифтом меншого розміру і розташовується нижче основної базової лінії тексту. Цей тег найчастіше застосовується для написання хімічних формул (наприклад, для цифр у формулі води) або математичних індексів.

Тег sup — це парний рядковий елемент, який використовується для відображення тексту у верхньому індексі. Він рендериться шрифтом меншого розміру і розташовується вище основної базової лінії тексту. Цей тег зазвичай застосовується для написання математичних степенів, порядкових числівників або посилань на виноски в тексті.

Тег code — це парний рядковий елемент, який використовується для позначення фрагмента комп'ютерного коду в тексті. За замовчуванням браузери відображають вміст цього тегу моноширинним шрифтом, де всі символи мають однакову ширину, що допомагає візуально виділити програмний код, назви змінних або файлів на фоні звичайної мови.

Тег pre (preformatted text) — це парний блочний елемент, який використовується для виведення тексту саме в тому вигляді, в якому він написаний у вихідному коді. На відміну від звичайних тегів, він зберігає всі пробіли, табуляції та розриви рядків, а текст усередині нього зазвичай відображається моноширинним шрифтом. Це робить його ідеальним для відображення прикладів коду, ASCII-графіки або будь-яких текстових структур, де форматування має ключове значення.
